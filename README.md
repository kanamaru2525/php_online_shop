# php_online_shop

### git_commands
リモート環境から変更を取得する
```
git pull
```
ファイルの登録
```
git add <ファイル名>
```
ファイルの変更や追加をコミット
```
git commit -m "コミットメッセージ"
```

ローカル環境の変更を確認する
```
git status
```
リモートにプッシュ
```
git push origin <ブランチ名>
```
